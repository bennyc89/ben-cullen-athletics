# Ben Cullen Athletics — Project Files

## Structure

### /site
- `bencullenathletics.html` — The main website. Single file, fully self-contained.
  Open in any browser. Upload to any web host.

### /demo
- `bca_site_demo.html` — Animated product demo walkthrough.
  Screen record this for Instagram reels or send to schools/clubs.

### /assets/posters
- International poster (HTML + PNG) — For Dubai/Middle East outreach emails.

### /assets/leaflets  
- Cobham leaflet front and back (HTML + PNG) — Send HTML to printer as A5 double-sided.

### /assets/banners
- Pull-up banner (HTML + PNG) — Send to banner printer as 850mm x 2000mm.

### /assets/instagram
- Story and post graphics ready to upload to Instagram.
